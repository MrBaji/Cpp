# Cpp - تمرین‌های برنامه‌نویسی پایه

این ریپازیتوری شامل تمرین‌های درس **مبانی برنامه‌نویسی** (دانشگاه شهید رجایی، رشته‌ی مهندسی کامپیوتر) است.

## محتوا
هر فایل (`t1.cpp` تا `t8.cpp`) یک تمرین جداست، با تمرکز بر:
- توابع بازگشتی (Recursive Functions)
- آرایه‌ها
- ساختارهای شرطی و حلقه‌ها

## نحوه‌ی اجرا
هر فایل با یک کامپایلر استاندارد C++ قابل کامپایل و اجراست:
```bash
g++ t1.cpp -o t1
./t1
